# Рекламные материалы AstroPulse

## Лист A4 «Карманный астролог»

Файлы:

- `flyer-a4.html` — макет для печати
- `flyer-a4.png` — готовая картинка A4 (2480×3508 px, ~300 dpi)
- `flyer-a4.pdf` — PDF для печати
- `qr-code.png` — QR-код на `https://t.me/AstroPulsee_bot`

### Как распечатать

1. Откройте `flyer-a4.html` в Chrome или Edge.
2. Нажмите **Печать** (`Ctrl+P`).
3. Формат бумаги: **A4**.
4. Включите **Фоновая графика** / **Background graphics**.
5. Масштаб: **100%**.

### Как сохранить в PDF

В окне печати выберите **Сохранить как PDF** вместо принтера.

## Визитка 90×50 мм «Карманный астролог»

Односторонняя визитка в том же стиле, что и флаер.

Файлы:

- `business-card.html` — макет для печати
- `business-card.png` — готовая картинка (1063×591 px, ~300 dpi)

### Печать визитки

1. Откройте `business-card.html` в браузере или используйте `business-card.png`.
2. Формат: **90×50 мм** (9×5 см).
3. Масштаб **100%**, включите фоновую графику.

## Лист A4 с визитками (10 шт.)

На одном листе **2×5 = 10 визиток** (90×50 мм).

Файлы:

- `business-card-sheet-a4.html` — макет для печати
- `business-card-sheet-a4.png` — готовая картинка A4 (2480×3508 px)
- `business-card-sheet-a4.pdf` — PDF для печати

### Печать листа

1. Откройте `business-card-sheet-a4.pdf` или `.png`.
2. Бумага **A4**, масштаб **100%**, фоновая графика включена.
3. Режьте по серым линиям между визитками.
