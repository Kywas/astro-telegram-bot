# Тест-пост 01 · Энергия дня

| Файл | Назначение |
|------|------------|
| `energy-day-base.png` | Реалистичное фото (основа) |
| `energy-day.gif` | GIF для Telegram |
| `post.txt` | Текст поста |
| `meta.json` | Настройки анимации |

## Опубликовать

1. Telegram → [AstroPulse Channel](https://t.me/AstroPulse_Channel) → новый пост
2. Прикрепи **`energy-day.gif`**
3. Вставь текст из **`post.txt`**
4. Кнопка: ссылка на @AstroPulsee_bot

## Пересобрать GIF

```powershell
python scripts/render_channel_gifs.py energy-day
```
