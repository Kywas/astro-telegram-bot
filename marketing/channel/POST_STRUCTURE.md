# Структура постов AstroPulse Channel

Канал: [@AstroPulse_Channel](https://t.me/AstroPulse_Channel) · бот: [@AstroPulsee_bot](https://t.me/AstroPulsee_bot)

---

## Формат одного поста

| Блок | Что писать | Длина |
|------|------------|-------|
| **1. GIF** | Реалистичное фото + лёгкое мерцание / дыхание света | 480×540 |
| **2. Заголовок** | Рубрика + эмодзи, цепляет за 3 сек | 1 строка |
| **3. Hook** | Боль или узнаваемая ситуация | 1–2 строки |
| **4. Тело** | Смысл простыми словами, без «100% предсказаний» | 3–6 строк |
| **5. Шаг на сегодня** | Один маленький шаг | 1–2 строки |
| **6. CTA** | Ссылка на бота (кнопка под постом) | 1 строка |

```
  post.txt           # готовый текст для Telegram
  energy-day-base.png
  energy-day.gif
```

---

## Публикация

**С GIF (вручную в Telegram):**
1. Открой канал → новый пост
2. Прикрепи `energy-day.gif`
3. Вставь текст из `post.txt`
4. Добавь кнопку → @AstroPulsee_bot

**Через бота (только текст + кнопка):**
```
/channelpost <текст из post.txt>
```
